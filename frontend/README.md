# Smart Movie Finder - Frontend

React-based UI for the Smart Movie Finder application.

## Installation

```bash
cd frontend
npm install
```

## Running the App

```bash
npm start
```

The app will run at `http://localhost:3001`

Make sure your backend API is running on `http://localhost:8000`

## Features

- Search movies using OMDb API
- Perform web searches using Serper API
- Scrape IMDb for detailed movie information
- Clean and responsive UI
- Modern purple gradient design
